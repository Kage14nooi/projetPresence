# Système de Gestion de Présence avec Hikvision

Plateforme web pour la gestion des présences étudiantes, intégrant le **terminal biométrique Hikvision** et le traitement des données Excel.

## Fonctionnalités

- Présence en **temps réel** via empreinte digitale Hikvision
- Import et manipulation des **données Excel** depuis l'appareil
- Notifications aux étudiants absents
- Filtrage par date et dashboard statistique
- Gestion centralisée des séances et étudiants

## Technologies

- **Backend :** Node.js, Express, Sequelize, Socket.io
- **Frontend :** React + TypeScript, TailwindCSS
- **Matériel :** Terminal biométrique Hikvision (empreintes + Excel)

> Deux modes de présence pour une **fiabilité et flexibilité** maximales.
