Pour résoudre les problèmes de lenteur ou de blocage de votre système, y compris Office, commencez par une simple réparation d'Office. Si le problème persiste, exécutez les utilitaires de résolution des problèmes de Windows, puis essayez de redémarrer votre ordinateur en mode sans échec. En dernier recours, si aucun de ces correctifs ne fonctionne, vous pouvez envisager de réinitialiser votre ordinateur. 
Étapes initiales (réparation et redémarrage)
Réparer Office :
Allez dans le « Panneau de configuration ».
Sélectionnez « Programmes » puis « Programmes et fonctionnalités ».
Cliquez avec le bouton droit sur Microsoft Office, puis choisissez « Modifier ».
Sélectionnez « Réparation en ligne » puis « Réparer » pour une réparation complète.
Redémarrer l'ordinateur :
Si l'ordinateur est bloqué, appuyez sur le bouton d'alimentation et maintenez-le enfoncé pour forcer son arrêt.
Attendez quelques secondes, puis appuyez à nouveau sur le bouton pour le rallumer. 
Étapes avancées (dépannage et réinstallation)
Exécuter les utilitaires de résolution des problèmes :
Allez dans « Paramètres », puis « Mise à jour et sécurité ».
Cliquez sur « Résolution des problèmes », puis sur « Autres utilitaires de résolution des problèmes ».
Exécutez l'utilitaire de résolution des problèmes pour « Applications du Windows Store » ou pour « Windows Update ».
Démarrer en mode minimal :
Recherchez et ouvrez l'outil de configuration du système (msconfig).
Dans l'onglet « Services », cochez la case « Masquer tous les services Microsoft ».
Cliquez sur « Désactiver tout ».
Dans l'onglet « Démarrage », désactivez tous les programmes.
Redémarrez l'ordinateur. Si le problème n'apparaît plus, réactivez les programmes un par un pour identifier le coupable.
Désinstaller et réinstaller Office :
Utilisez l'outil de suppression d'Office (Support and Recovery Assistant) de Microsoft pour désinstaller proprement votre suite Office.
Téléchargez et exécutez l'outil à partir de l'un des liens de Microsoft Learn ou de Microsoft Support.
Redémarrez votre ordinateur, puis réinstallez la suite Office. 
Dernière option (réinitialisation)
Si tout le reste échoue, vous pouvez réinitialiser votre PC. Cette option réinstalle le système d'exploitation, mais il est crucial de sauvegarder vos fichiers importants avant de procéder.
Pour plus de détails sur ce processus, consultez la documentation de votre fabricant d'ordinateur (par exemple, HP)
